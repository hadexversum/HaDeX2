# HaDeX2 1.0.0

This is a remastered version of HaDeX. It introduces new functionalities:
* hybrid-testing of differential data
* chiclet plot (single state and differential)
* butterfly plot (single state and differential)
* volcano plot
* manhattan plot
* uncertainty plot
* measurement variability plot (quality control feature)
* replicates plot (quality control feature)
* deuterium uptake aggregation (single state and differential)
* visualization on 3D protein structure
Dedicated, newly written GUI is in HaDeXGUI package.
