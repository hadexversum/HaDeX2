# Get residue positions

Get residue positions

## Usage

``` r
get_residue_positions(dat)
```

## Arguments

- dat:

  data imported by the
  [`read_hdx`](https://hadexversum.github.io/HaDeX2/reference/read_hdx.md)
  function.

## Value

a `data.frame` object.

## Details

Prepares data table for high-resolution values. It creates a long data
frame with each position in the sequence accompanied by the amino acid
placed in this position.
